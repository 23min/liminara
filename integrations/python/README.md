# Liminara Python SDK

Python SDK for Liminara reproducible nondeterministic computation with EU AI Act Article 12 compliance reporting.

## Install

```bash
uv sync --all-extras
```

## Usage

```bash
# Run the CLI
uv run liminara --help

# Run tests
uv run pytest

# Lint and format
uv run ruff check .
uv run ruff format --check .
```
